Panel Buttons

 Attributes: (* = required)

offImage* image to use when the button is NOT pressed

offAction function to call when the button is released

Dials (Knobs):

Dials consist of a "knob" image, a list of angles and optionally a list of labels for each angle.
The knob image should be pointing straight up (0º).  Angles are clockwise from 0º (straight up) to 360º.
If labels are specified, they should coordinate with the number of angles specified.
Clicking on the dial image will rotate the knob clockwise to the next angle and call the function specified by changeAction.


 HTML Tag:  <smac-dial>

 Attributes: (* = required) 
knobImage* image to use for the dial's knob
angles* angles in degrees for each dial stop (comma delimited)
labels labels that correspond to each angle (comma delimited)
changeAction function to call when the dial setting is changed
size percentage of browser width (default is 10)
currentIndex angle index to set knob (default is first angle)


<smac-dial knobImage = "Images/ui_Dial1.png" angles = "0,30,60,90,120,150,180,210,240,270,300,330" labels = "0,30,60,90,120,150,180,210,240,270,300,330" changeAction = "myDialChanged"> </smac-dial> <smac-dial knobImage = "Images/ui_Dial1.png" angles = "45,90,135" labels = "Low,Med,High" size = "10" changeAction = "myDialChanged"> </smac-dial>
